FIN-Vorkurs
===========

Installation
-------------

1. ```git clone git://github.com/renemeye/finvorkurs.git```
2. ```cd finvorkurs```
3. ``` bundle ```
4. Install [redis](http://redis.io) Server
	* On MacOS:
		```
		foo$ brew install redis
		```
5. start redis
		```
		foo$ redis-server
		```
6. ```rails s```
