FIN-Vorkurs
===========

Installation
-------------

1. git clone git://github.com/renemeye/finvorkurs.git
2. cd finvorkurs
3. bundle
4. install [redis](http://redis.io)
	* On MacOS:
		```bash
		brew install redis
		```
5. start redis
		```bash
		redis-server
		```
6. rails s
